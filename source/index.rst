===============
Open Vision API
===============

:website: https://openvisionapi.com
:source code: https://github.com/openvisionapi

Open Vision API is an open source computer vision API based on open source models.

Features
--------

* Support multiple hardware accelerator
* Support multiple deep learning frameworks
* Based on open source models
